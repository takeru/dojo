## 要点（ドキュメント形式用）

### 更新コマンド

Rustを最新バージョンに更新するのは簡単です：

```bash
rustup update
```

これだけで、rustc、cargo、標準ライブラリなど全部まとめて更新されます。

### どれくらいの頻度で更新する？

- Rustは**6週間ごと**に新バージョンがリリースされます
- 大きな破壊的変更は少ないので、定期的に更新してOK
- 新機能や性能改善が入るので、更新はおすすめ

### バージョン確認

現在のバージョンを確認するには：

```bash
rustc --version
cargo --version
rustup --version
```

## 対話形式の教え方ガイド（先生用）

### 導入

「Rustの新しいバージョンが出たらどうやって更新するか、知っておくと便利じゃぞ」

### 説明の流れ

1. **更新コマンドを紹介**
   ```bash
   rustup update
   ```
   「これだけで全部更新されるのじゃ」

2. **実際に試してもらう**
   「今のバージョンを確認して、updateを実行してみるのじゃ」

3. **頻度を説明**
   「6週間ごとに新バージョンが出るが、壊れることは少ないので安心して更新してよいぞ」
