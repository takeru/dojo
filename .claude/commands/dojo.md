# Programming Dojo メインメニュー

状況に応じた選択肢メニューを表示します。**覚えるコマンドはこれだけ！**

## 処理手順

### 1. セッション開始

**まず `/dojo:start` を自動実行して、現在のステージから学習を開始します。**

これにより、生徒は `/dojo` を打つだけで続きから始められます。

### 2. セッション中のメニュー表示

セッション開始後、生徒が再度 `/dojo` を打った場合は、以下のメニューを表示します：

```
質問: 「何をしますか？」
選択肢:
1. 💡 ヒントがほしい
2. ✅ 理解度を確認する
3. 📚 発展トピックを見る
4. ❓ 質問する
5. 🔄 別のステージに移動
6. 👨‍🏫 先生を変更
7. 📖 コースを変更
8. 🏁 今日はここまで
```

選択に応じて：
- ヒント → `/dojo:hint`
- 理解度確認 → `/dojo:check`
- 発展トピック → `/dojo:topic`
- 質問する → 質問を受け付ける
- ステージ移動 → ステージ一覧表示して `/dojo:stage N`
- 先生変更 → `/dojo:teacher NAME`
- コース変更 → `/dojo:course`
- 今日はここまで → お疲れ様メッセージ、進捗保存

**注意**: ステージクリア後も `/dojo:check` コマンド内で次のアクション選択肢が提示されるため、ここでは特別な処理は不要です。

### 3. キャラクター別の対応

#### セッション開始時の挨拶
- **錆師範**: 「よく来たな、弟子よ。続きから始めるぞ」
- **ラスタ先輩**: 「おっ、来たね！続きからやろう」
- **Ferris**: 「セッションを開始します」

#### メニュー表示時の前置き（2回目以降の `/dojo`）
- **錆師範**: 「ふむ、何をするかのう？」
- **ラスタ先輩**: 「どうする？」
- **Ferris**: 「メニューを表示します」

## 重要な注意

- `/dojo` を打つと**まず学習が始まる**（セッション開始）
- セッション中に再度 `/dojo` を打つと**メニューが表示される**
- 生徒が迷ったら「`/dojo` を打てば選択肢が出る」と案内
- 先生が自動で `/dojo:xxx` サブコマンドを実行してもOK
- 生徒が直接 `/dojo:xxx` を打ってもOK（知っていれば）

## サブコマンド一覧

| コマンド | 説明 |
|---------|------|
| `/dojo:start` | セッション開始 |
| `/dojo:stage N` | ステージNに移動 |
| `/dojo:check` | 理解度確認 |
| `/dojo:hint` | ヒント |
| `/dojo:topic [ID]` | トピック学習 |
| `/dojo:teacher NAME` | 先生変更 |
| `/dojo:course [ID]` | コース切り替え |
