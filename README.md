### Rust基礎コース（rust-basics）

Rustの基本構文を6ステージで学びます。

| Stage | 内容 |
|-------|------|
| 1 | 環境構築（rustup, cargo） |

| Stage | 内容 |
|-------|------|
| 1 | 所有権（ownership, move） |
| 2 | 借用（&, &mut） |
| 3 | ライフタイム基礎（'a） |

**前提**: rust-basics完了

| Stage | 内容 |

| 3 | 時間計測（ベンチマーク） |

| 6 | スレッドとブロック |
| 7 | 行列乗算 |

## 発展トピック（サブクエスト）

各ステージに関連する**深掘りコンテンツ**があります。
必須ではありませんが、実務で役立つ知識が身につきます。

**例**: rust-basicsコース
- `rust-toolchain` - rustup/rustc/cargoの関係
- `cargo-internals` - cargo runの内部動作
- `release-build` - デバッグ vs リリース
- `macros-intro` - マクロ入門

トピックは2つの方法で学べます：
1. **対話形式** - 先生と一緒に（5-15分）
2. **ドキュメント形式** - 要点だけ読む（2-5分）

## コース作成者向け

### コンテンツジェネレーター

新しいコースを簡単に作成できるツールがあります。

```
/dojo:content-generator
```

#### 機能

| モード | 説明 |
|--------|------|
| 新規コース作成 | 対話形式でコース全体を生成 |
| コンテンツ編集 | 既存ステージ・トピックを修正 |
| トピック追加 | 既存コースに発展トピックを追加 |
| バリデーション | フレームワーク要件への準拠チェック |

#### 使用例

**新規コース作成**（対話形式でガイド）:
```
/dojo:content-generator
→ 「新規コース作成」を選択
→ コースID、ステージ構成などを質問に答える
→ 自動でカリキュラム・トピック・course.jsonを生成
```

**既存コンテンツの編集**:
```
/dojo:content-generator "rust-basics の stage 2 の演習課題をもっと具体的に"
```

**バリデーション実行**:
```
/dojo:content-generator --validate rust-basics
```

#### 生成されるファイル

```
courses/{course-id}/
├── course.json           # コースメタデータ
├── curriculum/
│   ├── stage1-xxx.md     # 各ステージのカリキュラム
│   ├── stage2-xxx.md
│   └── ...
└── topics/
    ├── topic-a.md        # 発展トピック
    └── ...
```

詳細は `COURSE_AUTHORING_GUIDE.md` を参照してください。
